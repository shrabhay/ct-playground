CRM Record Push
Push a Salesforce or HubSpot customer record into CleverTap as a user profile with custom field mapping.
{ "d": [{ "type": "profile", "identity": "rahul@corp.com", "profileData": { "Name": "Rahul Verma", "Email": "rahul@corp.com", "Phone": "+919988776655", "CRM Stage": "Qualified Lead", "Account Manager": "Sneha Patel", "Deal Value": 120000 } }] }

Server-side Event
Fire a backend event — like Order Delivered or Payment Confirmed — that originates from your server, not the browser.
{ "d": [{ "type": "event", "evtName": "Order Delivered", "evtData": { "order_id": "BZ-20251234", "delivery_time": "2 days", "delivery_partner": "Delhivery", "rating_prompt": true }, "identity": "rahul@email.com", "ts": 1710000000 }] }
